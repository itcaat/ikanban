# Dopamine Pit -- Kanban Survivor

Браузерная игра-симулятор в виде канбан-доски. Задачи сыпятся на доску с нарастающей скоростью -- перетаскивай их через колонки, закрывай вовремя, не дай бэклогу взорваться.

**[Играть](https://itcaat.github.io/dopamine-pit/)**

## Как запустить локально

```bash
npm install
npm run dev
```

## Стек

React + TypeScript + Vite + Zustand + @dnd-kit + Framer Motion + Tailwind CSS
